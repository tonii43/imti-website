# Beranda Image Checklist

Put all Beranda images in this folder: `public/home/`

Use these exact filenames:

- `avatar-1.png`
- `avatar-2.png`
- `hero-slide-1.png`
- `hero-slide-2.png`
- `newsletter-cover.png`
- `instagram-preview.png`
- `tentang-logo.png`
- `menu-publikasi.png`
- `menu-administrasi.png`
- `menu-advokasi.png`
- `menu-keilmiahan.png`
- `menu-data-internal.png`
- `kontak-background.png`
- `kontak-photo.png`

Notes:
- Keep extension as `.png` (or update the code if you prefer `.jpg`/`.webp`).
- File names are case-sensitive in many deployment environments.
